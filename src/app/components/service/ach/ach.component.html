EASY ACH FILE CREATOR

Easy ACH File Creator is a standalone software program that allows you to create NACHA formatted ACH files using data you enter or migrate from any Excel or Comma-separated Values (CSV) file. The program supports multiple common Standard Entry Class Codes (SECs) and works independently without the need for any additional software.

Easy ACH File Creator creates a NACHA formatted file from data you enter or create in an Excel or Comma-separated Values file (CSV), guaranteeing it will work with any accounting package that can create a CSV file.
